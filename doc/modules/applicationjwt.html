createSession

  • createSession<AppSession, Secret>(appID: string, appLogin: string, secret: Secret, connector: Connector<AppSession, Secret>): Promise<object>
    • Defined in ApplicationJWT.ts:76

    Create a DataPeps session and an application session.

    return(p)

    On success the promise will be resolved with the DataPeps session, the application session and a boolean that indicates if its the first connection of the user to the DataPeps service. On error the promise will be rejected with an {@link Error} with kind:

    • ApplicationInvalidToken if the JWT token returned by the connector is invalid.
    • BadSecret if the secret of the user is wrong.
    • ApplicationConfigNotFound if the appID is not configured.

    Type parameters

    • AppSession

    • Secret: string | Uint8Array

    Parameters

    • appID: string

      The identifier of your DataPeps application.

    • appLogin: string

      The login of the identity in the referential of the application.

    • secret: Secret

      The secret of the DataPeps identity.

    • connector: Connector<AppSession, Secret>

      The JWT connector of your application
